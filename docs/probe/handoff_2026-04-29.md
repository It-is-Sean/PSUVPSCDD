## Non-negotiable interpretation rules

- ScanNet processed data already uses `frame_skip=20`; intended local-overlap experiments should use `scannet_max_interval=1`.
- Older ScanNet runs with `max_interval=30` are interval-confounded and should not support K-view claims.
- Old SCRREAM numbers are invalid as formal evidence because the local data was the released eval subset, not the full training-scale dataset.
- Symmetric CD alone is insufficient. Use fixed-sample robust evaluation: pred→GT precision side, GT→pred recall side, F-score thresholds, trimmed CD, and representative renders.
- The next planned training data direction is InteriorGS-style high-quality indoor 3DGS data on this server.
